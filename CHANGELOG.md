# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),

## [1.3.0] - 2026-05-06

### Added
- Lift angle management page for maintaining manufacturer/model lift-angle values.
- Timing calculator improvements tied to lift-angle data management.
- README updates to reflect latest setup and feature set.

### Changed
- Manufacturer lookup normalization for lift-angle matching to reduce duplicates and improve hit rate.
- Version metadata bumped to `1.3.0` in `package.json` and `package-lock.json`.
